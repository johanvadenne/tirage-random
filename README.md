# FR: Application de Tirage au Sort

Ce dépôt présente une application de tirage au sort de personnes développée en utilisant différents langages de programmation. L'application permet de sélectionner aléatoirement une personne parmi une liste de participants.

## Langages de Programmation

L'application de tirage au sort a été développée en utilisant les langages de programmation différents:

1. **Python**

2. **Ruby**

3. **C#**
